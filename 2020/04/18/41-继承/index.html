继承

发表于 2020-04-18 更新于 2020-04-19 分类于 JavaScript 阅读次数： Valine：
2.1k 2 分钟

下面介绍继承的几种方式

1. 原型链继承

通过prototype实现继承

引用类型的属性被所有实例共享
在创建 Child 的实例时，不能向Parent传参

2. 借用构造函数继承

通过call绑定this实现继承。

避免了引用类型的属性被所有实例共享
可以在 Child 中向 Parent 传参
方法都在构造函数中定义，每次创建实例都会创建一遍方法。

3. 组合继承

优点：融合原型链继承和构造函数的优点，是 JavaScript 中最常用的继承模式。
